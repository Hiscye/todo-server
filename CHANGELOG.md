## 📋 版本约定

### 分支命名
- 格式：`v1.X.Y-功能名`（如 `v1.3.0-ui-redesign`）

### 推送流程
```
1. 切到新分支：git checkout -b v1.3.0-ui-redesign
2. 实现功能 + 加版本注释头
3. 提交：git add . && git commit -m "feat: ..."
4. 推送：git push -u origin v1.3.0-ui-redesign
5. Railway 切到该分支部署
6. 测试通过后合并到 main
```

### 回滚方案
- 当前 Railway 部署的是 `main` 分支（v1.0.0）
- 任何版本出问题，Railway 切回 main 即可
- 所有历史版本都在 Git tag 里
