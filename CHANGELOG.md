### 📊 排序系统
- ✅ 7 种排序方式：
  - 📅 最新创建 / 最旧优先
  - ⏰ 截止最近 / 截止最远
  - 🔥 优先级高→低 / 低→高
  - 🔤 标题 A-Z
- ✅ 排序与筛选 / 搜索 / 标签筛选可叠加

## 📋 版本约定

### 分支命名
- 格式：`v1.X.Y-功能名`

### 推送流程
```
1. 切到新分支：git checkout -b v1.4.0-fancy-tags
2. 实现功能 + 加版本注释头
3. 提交：git add . && git commit -m "feat: ..."
4. 推送：git push -u origin v1.4.0-fancy-tags
5. Railway 切到该分支部署
6. 测试通过后合并到 main
```

### 回滚方案
- 当前 Railway 部署的是 `main` 分支（v1.0.0）
- 任何版本出问题，Railway 切回 main 即可
